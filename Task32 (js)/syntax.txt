VARIABLES AND DATATYPE!!!!!!!

variable is like a container to a store data.
var- older way, but can change the value.
let- can change the value.
const- cannot change the value


#var
var → (old, avoid using now)
var age = 25;
You can change the value

#let
let → (most commonly used)
let city = "Mumbai";
city = "Delhi";    // can change it
Can change the value

#const
const → (for constants)
const country = "India";
country = "USA";  // Error! can't change a const
Value cannot be changed
